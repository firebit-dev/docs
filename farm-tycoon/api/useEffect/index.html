useEffect

Similar to React's useEffect, this function runs a function when the component mounts and when the dependencies change.

Usage example:

local clicks = State.new(0)

local textLabel = Instance.new("TextLabel")
local button = Instance.new("TextButton")

myComponent.Mounted:Connect(function()
	myComponent.cleanup:Add(useEffect(function()
		textLabel.Text = tostring(clicks:Get())

		return function()
			-- Cleanup code here
			clicks:Set(0)
		end
	end, { clicks }))

	myComponent.cleanup:Add(button.Activated:Connect(function()
		clicks:Set(clicks:Get() + 1)
	end))
end)
